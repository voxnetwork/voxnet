# DEPENDENCIAS
npm install express
npm install mysql2

Verifique se o arquivo **package.json** está presente dentro das pastas **server** e **web**, pois é ele que permite restaurar o `node_modules` com `npm install`.


## 1. Instalar dependências do back-end
Acesse a pasta `server` e instale os módulos:
cd server
npm install

## 2. Instalar dependências do front-end
Acesse a pasta `web` e instale os módulos:
cd web
npm install

## 3. Iniciar o back-end
Dentro da pasta `server`:
npm start

## 4. Iniciar o front-end
Dentro da pasta `web`:
npm start
